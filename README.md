# 📦 MASA Inventory Pro
**MASA Inventory Pro** is a high-performance, professional-grade Inventory Management System built with C# and SQLite. It features a stunning, modern dark theme powered by Guna UI2, providing a seamless and premium experience for managing stock, sales, and data analytics.
---
## 🚀 Key Features
- **📊 Advanced Dashboard**: Real-time business analytics with gradient charts and live statistics.
- **📦 Product Management**: Full inventory control with barcode support, category filtering, and stock alerts.
- **💰 Sales Tracking**: Modern order processing with real-time stock reconciliation and history management.
- **🧹 Mass Actions**: Innovative "Delete All" functionality for bulk data resets with security confirmations.
- **✨ Premium UI**: Royal Purple dark theme, smooth fade animations, and a professional splash screen.
- **💾 Secure Data**: Automated local SQLite database with built-in backup tools.
---
## 🛠️ Technology Stack
- **Language**: C# (.NET Framework 4.7.2)
- **Database**: SQLite (Local, high-speed storage)
- **UI Framework**: Guna UI2 WinForms (Premium Design System)
- **Architecture**: N-Tier Architecture (BLL, DAL, Presentation Layers)
